
The **Discussion Thread System** allows users to create discussion threads, view individual posts, add comments, and reply to comments in a nested structure.

**Features**:
- Create discussion threads
- View all threads on the feed page
- View a single post with title and content
- Add comments to posts
- Reply to existing comments (nested replies)
- Clearly indicates reply context (e.g., “Replying to this comment”)
- Visually indented nested comments
- Timestamps for posts and comments
- Delete discussion threads


**Tech Stack Used**:
<br/>
Frontend
- React.js
- CSS

Backend
- Node.js<br/>
- Express.js<br/>
- MongoDB
<br/>
**Setup**:

1. Clone the Repository
```bash
git clone https://github.com/ArjunMohanty/Discussion-Thread-System.git
cd Discussion-Thread-System
```

2. Backend Setup
```bash
cd backend
npm install
node server.js
```
3. Frontend Setup
```bash
cd frontend
npm install
npm start
```
